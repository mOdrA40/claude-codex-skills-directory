# Frontend Performance Mastery (Senior → Principal)

## Operate

- Confirm performance target: route load, LCP, INP, hydration, mobile device responsiveness, or long-session memory stability.
- Measure first, then optimize the bottleneck with the highest user-visible cost.
- Keep budgets and regression visibility explicit.
