# SvelteKit Mastery (Senior → Principal)

## Operate

- Confirm SvelteKit version, adapter target, SSR/CSR/SPA expectations, form and action complexity, auth model, cache/fetch strategy, and deployment platform.
- Separate route loading, actions, UI state, and backend integration clearly.
- Optimize for web fundamentals: simple data flow, progressive enhancement, and predictable failure handling.
