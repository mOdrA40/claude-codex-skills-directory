# Angular Mastery (Senior → Principal)

## Operate

- Confirm Angular version, standalone adoption, routing model, RxJS complexity, state-management approach, SSR needs, and enterprise governance constraints.
- Separate presentation, orchestration, services, and shared platform concerns clearly.
- Optimize for maintainability under large-team and long-lived application pressure.
