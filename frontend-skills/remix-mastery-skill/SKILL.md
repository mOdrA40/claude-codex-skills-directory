# Remix Mastery (Senior → Principal)

## Operate

- Confirm Remix version, runtime target, auth model, nested route complexity, mutation flow, and deployment platform.
- Separate loaders, actions, route UI, and backend boundaries clearly.
- Optimize for predictable web behavior, progressive enhancement, and explicit route ownership.
